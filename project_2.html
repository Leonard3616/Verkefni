Project 2

Computer-aidded cutting

Introduction

This second project will be divided in 2 parts:

The first part of the proyect is a vinyl cutter used to create a sticker, the dimensions are 100x50mm

The second part will be compoused of a parametric design, press-fit model of building blockes. This model must be scalable in different ways, as thickness, heights and widths.

Also a small group project, finding the kerf for the laser-cuuter used for the part 2 of this project

Vinyl cutter

For this first part of the project will be learned how to cut a sticker with the use of a vinyl cutter.

I started to determinate which sticker to create. I choose to make an sticker of Droopy the cartoon character, because U watch it very often when I was a kid

After deciding the sticker model I prooced to download a image in a .png form, then was imported to Inkscape to make the neccesary changes to create the sticker

To start editing this image, I first change the page measurments in properties to 100x50mm. So that will be eassier to have the right size for our image. Then we will go to Path/Trace bitmap press apply and separate the original image from the trace bitmap one. The original image can be discart now.

Now for last we have to select our image, go to fill and stroke options and select Fill/No paint, Stroke paint/Flat colour, Stroke style/Width=0,02mm. This last measurment is for the vinyl cutter.

Now to finish this part we have to send this modifyed image to the vinyl cutter. Fill the vynil cutter with the sticker colour and press print.

Last part is just to plug the small parts of your new sticker and place it wherever you want.

Laser cut

It took me quite a while to decide what to design. I tried to design something that I will be able to use and I came with an idea of a liquor shelve. I drew it more or less as it should look.

Then I missunderstood about the dimesions of the material that we will use for this, thats why the dimensions are quite longer then it shoudld be in the picture above. The correct dimmensions are 500x500mm

To design this we will use Fusion360, we will also design it with parameters, so this model will be scalable in different ways, as thickness, heights and widths.

Design in fusion 360:

To start drawing in Fusion360 with finger joints and different connectors. We will create parameters to help us with this. There is a video that helped the most to learn how to use this parameters and create a model wich is this one "Fusion360: Automatically sized finger-joints". So with the help of this video I started the design with my parameters.

There was no need of many parameters in my design because it wasn't complicated. I started with the back side added finger joints in the sides but the top, and decide to add 2 holes in the corner which I will use to hang the liquer shelve on the wall.

So as following the video mentioned before I started adding the sides. And also connect them with the finger joints from the back side. I needed also to create 2 holes on the sides where will be hanging the flat parts of the shelve. To make this work I had to add a new fitting on the sides and use hinger joints as holes on the sides. SO I was able to place the flat side. In my desing there are 2 flat sides so that the first one will hold the bottles, and the second one will hold wine glases.

So to finish the model I nessured the bottom of the wie glasses and create some holes in the bottom flate part so that they will hold it. Then I place the front part and it was done. Because of the 500x500mm size of the design, this will be able to hold 3 bottles and 3 wine glasses.

Here you can download this model: grabcad

The next step now will be the small group project to calculate the kerf. I made this group project with Ayush-tseren. We cut a test with the laser cutter. This little test consisted in many squares. Then after the machine (Fusion edge 24) was done, ee meassure the gaps between the squares and we had a result for the kerf of 0,1313mm which was different from the other groups. But I used a kerf of 0,14mm which end up being very good .

When this was done I procced with the editing of the model in Fusion360 to add our kerf and laser cut my model. For this I followed this video Fusion360: DXF export with kerf compensation for laser cutting.

Then I prepeared it to exported as a .DXF document wich will allow us to work with it in inskape. Innkscape will be use to edit our DXF. document to get it ready to be laser cut. This videos are the ones I followed for this:Fusion 360 - 14 - Exporting to DXF - SE21 and DXF Segmented Line Fix in Inkscape 2

Before the laser cut we were told to lower the cut velcity to 15%. I placed a acryl in the "Fusion edge 24", opened the air valve and press ad print and wait fot it to be done.

To finish this project I assembled the liquor shelve and test it.
